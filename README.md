# MCFNL Course, part II

This repository will be used for the practical part of this course.

## Requirements

To follow this part you will need the following (assuming you use Windows and you bring your own laptop):

- Install git: <https://git-scm.com/install/>
- Install python: <https://www.python.org/downloads/>
- Install an IDE. The one recommended for this course is VS Code: <https://code.visualstudio.com/>. You can use a different IDE but no support will be provided.

You will also need to do the following:

1. Create a GitHub user at: <https://github.com/>
2. Form a team with other two students.
3. One of the members of the team must create an issue in this repo indicating the team's name and mentioning all the team members.

## Evaluation

Grades will be awarded based on the following:

- 70% from the final project, made by the team. You can find some project proposals at the end of the lecture notes.
- 30% on everyday's work: completing the assigned tasks, merging pull requests (PRs) which pass automatic tests, assistance, etc.
